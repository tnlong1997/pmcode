# pmcode
Backend for Primor
---------

### In repo command:

* Local lint test:
	...in pmcode directory:
	'''
	npm run lint
	'''

* If there is some lints, auto fix by using:
	...in pmcode directory:
	'''
	npm run lint-fix
	'''


* Install git pre-push hook:
	...in pmcode directory:
	'''
	./scripts/install-hooks.bash
	'''
